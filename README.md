Skyline is a UWP weather application.

## Features:

- Real time weather
- Daily forecast (7 days)
- Hourly forecast (24 hours)
- Minute level precipitation forecast (2 hours)
- Weather warning
- 14 life indices
- Sunrise and sunset time
- Daily weather/warning notifications
- Live tiles
- Lock screen weather
